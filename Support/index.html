Thanks for purchasing this KodeTravel Tourism html5/css3 theme

KodeTravel tourism theme is for travel agency and can be used for trip planner organizations. This template gives simple and elegance look and feel no messy layout with overflow of content. KodeTravel Tourism HTML Theme is designed to help organizations or individuals as travel agents, trip advisor, and trip planner. Customization is fun to do with this template, everything is managed and well commented. If you feel any issues please read this documentation and you will get to know everything, if you still need any help or you want me to customize it for you, you may send me an email at support@kodeforest.com.

Understanding Grid System

Back to Top

This template use bootstrap grid to make it very easy for anybody to understand the grid system. There is some basic details about its html structure below.

Parent Classes of Sections

.kd-serviceinfo class is created parent class over services, .kd-blog-list .kd-bloggrid class created for Blog Grid, .kd-blogmedium class created for Blog Medium. .kd-package-list created for package listing.#mainbanner created for main slider.

A little bit step onto innovation

.kd-subheader class we have created this class to manage sub header which is coming under menu bar it take background and its position we have used bootstrap classes for child elements to make it maximum mobile friendly / Responsive.
